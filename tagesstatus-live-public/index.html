Eigenständige Version · keine Daten ohne hinterlegte Keys
Diese Seite ist nicht mit einer Umgebung verbunden und enthält keine eingebetteten Keys. Beim ersten Öffnen fragt sie deine Tokens ab; sie werden nur lokal im Browser (localStorage) gespeichert. Ohne hinterlegten Key zeigt der jeweilige Abschnitt „keine Daten". Daten werden direkt per Browser-Abruf bei den Anbietern geholt — das funktioniert nur außerhalb eingeschränkter Sandboxes (also als gehostete/lokale Datei).
Sicherheit: Keys liegen im Klartext im Browser dieses Geräts. Manche Anbieter (OpenAI, Anthropic, Tailscale) blockieren Browser-Abrufe per CORS bzw. raten von Client-seitigen Keys ab — dort kann statt Daten ein CORS-/401-Fehler erscheinen. Für solche Quellen ist ein kleiner Server-Proxy nötig.
